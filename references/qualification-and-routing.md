# Qualification and Routing

## Variables de calificacion
- Objetivo academico: tercer ciclo, bachillerato por madurez o por edad.
- Ventana de tiempo: inmediata, corta, media.
- Disponibilidad semanal: alta, media, baja.
- Barrera principal: costo, tiempo, confianza, dificultad academica.
- Intencion de compra: explorando, interesado, listo.

## Preguntas sugeridas (maximo 1 por turno)
- "Que programa te interesa sacar primero: tercer ciclo o bachillerato?"
- "Prefieres clases en vivo o estudiar a tu ritmo?"
- "Buscas empezar de inmediato o para una convocatoria especifica?"
- "Que te frena hoy para matricularte: tiempo, costo o modalidad?"

## Reglas de enrutamiento
- Si disponibilidad baja -> priorizar asincronico.
- Si necesita disciplina externa -> priorizar sincronico.
- Si intencion es alta -> ir directo a cierre con CTA de matricula.
- Si hay mucha incertidumbre -> demo + resumen breve de beneficios.

## Resultado esperado
Cada diagnostico debe cerrar con una recomendacion explicita y un paso siguiente claro.
