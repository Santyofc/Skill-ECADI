# Intent Map

## Intenciones principales
- info-general
- requisitos
- fechas-convocatoria
- costos-promociones
- modalidad-sincronica
- modalidad-asincronica
- matricula
- demo
- objecion-costo
- objecion-tiempo
- objecion-confianza
- seguimiento
- reactivacion

## Respuesta recomendada por intencion
- info-general -> resumen de valor + pregunta de calificacion.
- requisitos -> respuesta puntual + nota de vigencia.
- fechas-convocatoria -> fecha absoluta + confirmacion de vigencia.
- costos-promociones -> rango/resumen + confirmacion final en canal oficial.
- modalidad-* -> comparativo breve + recomendacion contextual.
- matricula -> pasos concretos + confirmacion de avance.
- demo -> oferta de demo + objetivo del demo.
- objecion-* -> aplicar VERA + micro-CTA.
- seguimiento/reactivacion -> mensaje corto + una accion.
