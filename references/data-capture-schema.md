# Data Capture Schema

## Campos minimos
- nombre (si disponible)
- programa objetivo
- modalidad preferida
- disponibilidad horaria
- barrera principal
- etapa de lead
- proximo paso acordado

## Campos de seguimiento
- fecha ultimo contacto
- estado actual
- nota de vigencia pendiente (si aplica)
- motivo de no cierre (si aplica)

## Regla
Capturar solo lo necesario para avanzar la conversacion con contexto y continuidad.
