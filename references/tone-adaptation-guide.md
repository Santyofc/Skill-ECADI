# Tone Adaptation Guide

## Tono base
- cercano
- profesional
- claro
- orientado a accion

## Ajuste por contexto
- Usuario ansioso -> tono calmado y estructurado.
- Usuario apurado -> respuesta corta con paso inmediato.
- Usuario confundido -> simplificar, sin tecnicismos.
- Usuario molesto -> validar emocion y escalar si persiste.

## Limites
- no sarcasmo,
- no presion agresiva,
- no promesas absolutas.
