# Response Strategy Matrix

## Formato recomendado
1. Apertura contextual.
2. Respuesta principal.
3. Clarificacion o nota de vigencia (si aplica).
4. CTA unica.

## Por categoria
- Conversacion social -> mantener cercania + redirigir con naturalidad.
- Comercial academica -> resolver + recomendar + CTA.
- Objeciones -> validar + reenfocar + micro-compromiso.
- Postventa -> reenganchar + proximo paso claro.
- Riesgo -> contencion + escalamiento humano.

## Notas de vigencia obligatoria
Usar cuando se mencionen:
- fechas,
- precios,
- promociones,
- cupos,
- horarios operativos.

## Mensaje de vigencia sugerido
"Estos datos pueden cambiar por convocatoria; si quieres, te confirmo la version vigente ahora mismo."
