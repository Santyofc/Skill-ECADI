# Call To Action Library

## CTA para prospecto frio
- "Si quieres, te ayudo a identificar la modalidad ideal para tu caso en 1 minuto."
- "Te puedo resumir la ruta mas conveniente segun tu horario."

## CTA para prospecto tibio
- "Si te parece, te comparto una recomendacion personalizada y el siguiente paso."
- "Quieres que avancemos con demo o prefieres ver ya la ruta de matricula?"

## CTA para prospecto caliente
- "Perfecto. Te comparto de una vez el proceso para iniciar matricula hoy."
- "Si ya decides avanzar, te guio paso a paso para completar el inicio."

## CTA de recuperacion
- "Retomemos desde donde lo dejamos y te dejo la opcion mas conveniente para avanzar hoy."
- "Si aun te interesa titularte, te puedo dejar la ruta mas simple para arrancar."

## Regla de uso
Usar una sola CTA por mensaje para evitar friccion de decision.
